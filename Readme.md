Commandline editor written in ruby

Alpha, lets see if this works...

Finished:

 - viewing / scrolling / editing / saving / creating
 - Home/End
 - basic Tab support (tab == 2 space)
 - change-indicator
 - writeable indicator
 - backspace / delete
 - find / go to line
 - delete line

Install
=======
    sudo gem install ruco

Usage
=====
    ruco file.rb
